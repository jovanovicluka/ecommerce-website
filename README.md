# Simple React + Zustand App

This is a minimal web application built with **React** and **Zustand** for state management.  
The app allows users to:

- Sign up for a new account
- Log in with existing credentials
- Log out securely
- Manage a shopping cart (add, remove, update items)

---

## 🛠 Tech Stack

- **React** – UI library for building components
- **Zustand** – Lightweight state management
- **JavaScript (ES6+)** – Core language
- **CSS** – Styling

---

## 🚀 Features

### Authentication

- **Sign Up**: Create a new account
- **Log In**: Access with existing credentials
- **Log Out**: End session and clear user data

### Cart Management

- Add items to cart
- Increase/decrease item quantity
- Remove items completely
- Cart state persists during session
